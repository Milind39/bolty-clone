# Test Prompts for Template Endpoint

## React Project Examples

```json
{
  "prompt": "Analyze this project: I'm building a React application with components for a shopping cart. I need to manage state with Redux and implement routing with React Router."
}
```

```json
{
  "prompt": "I'm working on a front-end web application using React, with multiple components, hooks, and context API for state management."
}
```

## Node.js Project Examples

```json
{
  "prompt": "Analyze this project: I'm creating a Node.js backend with Express.js, MongoDB for the database, and JWT authentication."
}
```

```json
{
  "prompt": "I need help with a server-side application built on Node.js and Express that handles API requests and interfaces with a PostgreSQL database."
}
```

## Modified Prompt Format (More Specific)

Try using this more specific prompt format:

```json
{
  "prompt": "Determine if this is a React or Node.js project: I'm building a React application with components for a shopping cart."
}
```

```json
{
  "prompt": "Determine if this is a React or Node.js project: I'm creating a Node.js backend with Express.js and MongoDB."
}
```
